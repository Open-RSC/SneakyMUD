# SneakyMUD

This project requires CMake (https://cmake.org/install/)

Obtaining and compiling the project:
```
git clone https://github.com/Open-RSC/SneakyMUD.git/
cd SneakyMUD
mkdir build
cd build
cmake ..
cmake --build .
```
